性能分析

​ 1.切换到项目目录执行,生成链接地址,得到性能图

$ npm run preview -- --report

​ 2.webpack 排除打包

vue.config.js

    externals:{
        'element-ui': 'ELEMENT',
        'xlsx': 'XLSX'
    }

​ 3.cdn 文件配置(注意顺序)

vue.config.js

    let cdn ={ css:[],js:[]}
    let externals = {}
    const isprod = process.env.NODE_ENV === 'production'
    if(isprod){
        externals:{
            'element-ui': 'ELEMENT',
            'xlsx': 'XLSX'
        }
        cdn = {
            css:[],
            js:[]
        }
    }

​ 3.注入 cdn 变量

    config.plugin('html').tap(args => {
        args[0].cdn = cdn
        return args
    })

public.index.html

    <% for(var js of htmlWebpackplugin.options.cdn.css)
        <link rel="stylesheet" href="<%=css%>" />
    %>

​ 4.在 nodejs 环境中应用并代理跨域

​ 第一步,建立 web 服务文件夹 hrserver

    mkdir hrserver
    cd hrserver

​ 第二步,在该文件夹初始化 npm

    npm init -y

​ 第三步,安装服务端框架 koa(也可以采用 express 或者 egg)

    npm i koa koa-static

​ 第四步,拷贝 dist 到 hrserver.public 下

​ 第五步,在根目录创建 app.js

    const Koa = require('koa')
    const app = new Koa()
    //koa中间件,需要放在静态化前面
    const { historyApiFallback} = require('koa2-connect-history-api-fallback')
    //koa跨域中间件
    const proxy = require('koa2-proxy-middleware')
    app.use(proxy({
        target:{
            '/prod-api/(.*)':{
                target:'代理地址/api',
                changeOrigin:true,
                pathRewrite:{
                    '/prod-api':""
                }
            }
        }
    }))
    app.use(historyApiFallback({
        //白名单,不处理
        whiteList:['/prod-api']
    }))
    app.use(server(__dirname + "/public"))
    app.listen(3333,()=>{
        console.log('项目已启动,访问地址http://localhost:3333')
    })
